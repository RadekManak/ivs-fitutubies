User Interface

Main application window interface is divided into four sections.

Menubar

At the top you can see menubar with options for opening this manual and closing the application.

Result display

Display shows result of calculation with precision of 8 decimal places.

Input line

Below, there is input line where expression that is to be solved should be inserted. Both keyboard and keypad can be used.

Keypad

In the lower part of the application there is a keypad with most of the implemented functions. See function reference for the complete list of functions.

Order of operations

Operations are evaluated in following order: Constants -> Functions -> Factorial -> Modulo -> Exponentiation -> Multiplication and division -> Addition and subtraction.

Reference

Addition

Use x + y to add numbers.

Subtraction

Use x - y to subtract numbers.

Multiplication

Use x - y to multiply numbers.

Division

Use x / y to divide numbers.

Exponentiation

x^y computes x to the power of y.

Root

  • Square root

    Square root can be computed by either sqrt(x) or root(x).

  • N-th root

    Root of any degree can be computed by root(degree:x)

Factorial

Factorial be computed by number! syntax.

Logarithm

  • Decadic

    Logarithm to the base 10 can be computed by log(x)

  • Real base

    Logarithm of any base can be computed by log(base:x). If natural logarithm is needed use log(e:x). This works because Euler's number is implemented constant.

Trigonometric functions

sin(x), cos(x) and tan(x) are implemented. They require number in degrees as input.

Modulo

Remainder after division can be computed with C-like syntax x%y.

Limitations

Brackets are not currently supported. They can only be used in functions such as sqrt(x).
